Features

Textual Comprehension Adjustment

The text transformation tool allows you to highlight any text and select an age from the slider. Then, after hitting the "send" button, the extension will automatically change the text to your liking.

Dynamic Dictionary

The dynamic dictionary offers a personalized definition to highlighted text. Then the extension will provide a definition to the highlighted word.

Problem Solver

The problem solver allows you to highlight a "problem" on a webpage. It will automatically provide you with a solution to the problem and provide detailed steps on how it reached its conclusion.

Translate++

The Translate++ allows you to translate words using cutting-edge translation technology. It allows you to select ANY langauge in the world and translates it using the utf-8 charset. Translate++ has been tested for many languages and was previously used to communicate with Japanese exchange students.
